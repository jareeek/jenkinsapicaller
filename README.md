Use Case: I as a QA Analyst want to run a custom workflow IF a build for certain platform with certain deploy type was succesfully cooked in 24 hours timeframe

Set JENKINS_URL, JENKINS_USER, JENKINS_SECRET as environment variables, modify the rest of params however your jenkins project requires (e.g. tree, xpath filter params etc.)
